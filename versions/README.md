# Edit Version Folders

This project keeps lightweight edit-version folders to make progress visible by folder name.

Rule:

- Folder format: `2026-04-22-git-hub-vX.Y.Z`
- One folder is created per meaningful editing batch
- Each folder stores a short summary and related commit hash

Example:

- `versions/2026-04-22-git-hub-v0.4.0`
